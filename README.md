## O PROJEKTU
Toto je prémiové rozhraní pro platformu **Beliansky.eu**. Postaveno na Next.js 14 s využitím architektury **L-CODE GUARDIAN**, která zajišťuje extrémní výkon (60 FPS), absolutní stabilitu a moderní "Digital Architect" estetiku.

## TECHNOLOGICKÝ STACK
- **Core:** Next.js 14 (App Router) / TypeScript
- **Styling:** Tailwind CSS / Custom L-CODE Engine
- **Security:** SecurityKernel v3.1 (Anti-Hydration / Goliáš Protocol)
- **Data:** Wix Headless Bridge (Dynamic CMS Mapping)
- **Performance:** Sharp Image Optimization / GPU-Accelerated Animations

## STRUKTURA
- `/src/app`: Logika routování a serverové komponenty.
- `/src/components`: Atomické UI entity v standardu Prismatic Silk.
- `/src/kernel`: Bezpečnostní jádro a ochrana klientské vrstvy.
- `/src/lib`: Wix Bridge a překladová matice (SK/EN/GA).

## VÝVOJ A DEPLOY
Pro lokální vývoj:
```bash
npm run dev
```

Pro produkční build:
```bash
npm run build
```
